# Legal and Policy Caveats

## What This Tool Does

This tool extracts publicly visible data from LinkedIn profile pages as rendered in your own browser session. It reads the DOM of a page you have already loaded and are viewing. It operates entirely client-side within your browser.

## What This Tool Does Not Do

- It does not use LinkedIn's official APIs.
- It does not automate LinkedIn account actions (logging in, sending messages, connecting, etc.).
- It does not bypass authentication or access controls.
- It does not make network requests to LinkedIn servers beyond what your browser has already done to load the page.

## User Responsibility

Users are responsible for ensuring their use of this tool complies with:

- **LinkedIn's Terms of Service** -- LinkedIn's ToS restricts certain forms of data collection and scraping. Review the current terms before use.
- **Applicable laws** -- Data protection regulations (GDPR, CCPA, etc.) may govern how you collect, store, and use personal data, even if publicly visible.
- **Your organization's policies** -- Your employer or institution may have policies about data collection tools.

## Intended Use

This tool is intended for individual, personal use cases such as:

- Exporting your own profile data for backup or portfolio purposes.
- Extracting a single profile's visible information for legitimate professional purposes.

## Not Intended For

- Mass scraping of LinkedIn profiles.
- Automated or scheduled data collection.
- Building databases of LinkedIn user data.
- Any use that violates LinkedIn's Terms of Service.

## Disclaimer

This tool is provided as-is. The authors make no guarantees about its legality in any specific jurisdiction or use case. Users assume all responsibility for how they use the extracted data.
